-----Strapi-----------------------

Iniciar Strapi en modo "watch" (modo desarrollo)

          npm run develop

Iniciar Strapi sin modo "watch" (modo producción)
          npm run start

Compilar el panel de administración de Strapi

          npm run build

Desplegar el proyecto Strapi

          npm run deploy

Sembrar la base de datos con datos de ejemplo
          npm run seed:example

Mostrar todos los comandos disponibles en Strapi

          npm run strapi


Para acceder a la api es la misma ruta agregando lo siguiente

/api/+ nombre del contenido

ejemplo home

/api/home

Pero sin dar permisos no va a funcionar

Hay que darle permisos /
